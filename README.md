# Interview Copilot — ClinTrialAI

Real-time interview assistant that listens to interviewer questions and prints answers instantly.

## Deploy to Render (Free)

1. Push this folder to a GitHub repository
2. Go to render.com and connect your GitHub repo
3. Add environment variable: `ANTHROPIC_API_KEY` = your key from console.anthropic.com
4. Deploy — get your live URL
5. Open on your phone in Chrome

## How it works

- Tap Start Session
- Hold MY TURN while YOU are speaking/presenting
- Release when interviewer asks a question
- Answer prints on screen in 1-2 seconds
